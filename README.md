---

## claude/skills/

Slash commands for [Claude Code](https://claude.ai/code). To use, copy the `.md` file into your project's `.claude/commands/` folder and invoke it as a slash command.

| Tool | Description |
|------|-------------|
| [compress.md](claude/skills/compress.md) | Compress, evaluate, compare, or repair AI instruction artifacts while preserving behavioral integrity and safeguards |
